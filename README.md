Movie Recommendation Engine in Python
Overview
This repository contains Python code for a movie recommendation engine. The engine uses the MovieLens 25M Dataset to recommend movies based on user ratings and similarities. It's designed to run in Google Colab for ease of use and accessibility.

Features
Data Upload: Code to upload the 'movies.csv' and 'ratings.csv' files to Colab.
Data Cleaning: A function to clean movie titles for better search and comparison.
Search Functionality: Using TF-IDF Vectorizer and Cosine Similarity to search for movies.
Recommendation Algorithm: Creating a recommendation engine based on user ratings and movie similarities.
Interactive User Interface: Using IPython widgets to create an interactive search and recommendation display.
Installation
To use this project:

Open the notebook in Google Colab.
Upload the required 'movies.csv' and 'ratings.csv' files from the MovieLens 25M Dataset.
Usage
Data Preparation: Run the cells to upload the datasets and preprocess the data.
Search for Movies: Type a movie title into the interactive text box to search for movies.
Get Recommendations: The engine will display the top 10 recommended movies based on the selected movie.
